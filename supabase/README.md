# Миграции Supabase

Файлы выполняются строго по имени:

1. `202607010000_baseline.sql` — исходные профили, сайт, NFC и Storage;
2. `20260705_story_submissions.sql` — пользовательские истории;
3. `202607150001_ecosystem_core.sql` — identity, сезоны, геймификация, события и базовый YouGile;
4. `202607150002_full_platform.sql` — организаторы, очереди, безопасность, vault и полный API.

Для новой базы применяйте все четыре. Для действующей базы baseline уже был выполнен как `schema.sql`; повторный запуск в основном идемпотентен, но безопаснее отметить его применённым и выполнить только следующие миграции.

После миграций проверьте:

```bash
curl https://BACKEND_URL/ready
```

Ответ должен содержать `"ready": true`.

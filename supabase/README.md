# Миграции Supabase

Файлы выполняются строго по имени:

1. `202607010000_baseline.sql` — исходные профили, сайт, NFC и Storage;
2. `20260705_story_submissions.sql` — пользовательские истории;
3. `202607150001_ecosystem_core.sql` — identity, сезоны, геймификация, события и базовый YouGile;
4. `202607150002_full_platform.sql` — очереди, безопасность, vault и полный API;
5. `202607160003_beta_core.sql` — финальная Telegram-only схема закрытой беты.

Для полностью новой пустой базы применяйте все пять. Для действующей базы сайта, где уже есть `profiles`, контент и истории, не повторяйте первые два legacy-файла: запускайте только файлы 3–5.

После миграций проверьте:

```bash
curl https://BACKEND_URL/ready
```

Ответ должен содержать `"ready": true`.
